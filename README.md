# JavaChatServer

JavaChatServer est un programme console (Command Line Interface) écrit en Java permettant la communication textuelle entre plusieurs clients connectés au serveur via le protocole TCP et le programme client correspondant (voir JavaChatClient - Client de messagerie directe avec interface graphique : https://github.com/DevBlocks42/JavaChatClient).

Le fonctionnement du programme a été testé sur les systèmes d'exploitations suivants : Ubuntu 22.04 (Desktop & Live Server) et Windows 10. Néanmoins, celui-ci devrait pouvoir être exécuté sur n'importe quel système disposant de Java.
